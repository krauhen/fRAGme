Usage
============

Use as package from cli
-------------

.. code-block:: bash

    $ python -m venv venv
    $ source venv/bin/activate
    (venv)$ pip install fRAGme
    # set OPENAI_API_KEY in .env to "YOUR_OPENAI_API_KEY"
    (venv)$ uvicorn src.fRAGme.app:app --host 0.0.0.0 --port 8080
